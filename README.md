Git Commands
=============

gitt init - initializing git respository
git add - adding file or directory inside the git
git status - indentify the current state of the work done so far
git commit -m "message" - commit your changes
git rm - cashed filename - remove added file

Configure proxy
=================
git config --global http.proxy proxyaddress
git config --global https.proxy proxyaddress

Configure username and password
================================
git config --global user.name githubusername
git config --global user.password githubusername
